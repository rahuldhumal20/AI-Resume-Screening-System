# AI Resume Screening System 🚀

## 📌 Overview
This project is an AI-powered Resume Screening System built using LangChain and Groq API.

It evaluates candidates based on job descriptions and provides:
- Skill matching
- Scoring (0–100)
- Explainable results

## ⚙️ Tech Stack
- Python
- LangChain
- Groq API
- LangSmith (Tracing)

## 🔄 Pipeline
Resume → Extraction → Matching → Scoring → Explanation

## 🧠 Features
- Skill extraction
- Intelligent matching
- AI-based scoring
- Explainable outputs
- LangSmith tracing

## 📊 Results
- Strong Candidate → High Score
- Average Candidate → Medium Score
- Weak Candidate → Low Score

## 🚀 How to Run

```bash
pip install -r requirements.txt
python main.py
